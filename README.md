# 📄 Letter Calculator App

## 🚀 Overview

This is a simple React application that counts the number of letters entered in an input field.

---

## ✨ Features

* Enter any text in the input field
* Instantly displays the number of letters
* Dynamic UI update using React state

---

## 🛠️ Technologies Used

* React
* JavaScript
* CSS

---

## 📂 Project Structure

```
src/
 ├── component/
 │    └── LetterCalculator/
 │         ├── index.jsx
 │         └── index.css
 ├── App.jsx
 ├── App.css
 ├── main.jsx
```

---

## ▶️ How to Run the Project

1. Install dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```

3. Open in browser

```
http://localhost:5173
```

---

## 📌 Usage

* Type any text in the input box
* The app will show the number of letters entered
